Proposed Extensions
===================

Fluid Boundary Conditions
-------------------------


T-infinity defines the following boundary conditions:
    - Farfield
    - Constant Temperature No-Slip
    - Adiabatic No-Slip
    - Tangent Slip
    - Cartesian Plane Symmetry (x, y, z)
    - Arbitrary Plane Symmetry
    - Supersonic Outflow
    - Subsonic Outflow
    - Supersonic Inflow
    - Subsonic Inflow
    - Forced Overset Interpolation
